# 💜 MFP 2026 — Maratona Feminina de Programação

Página web desenvolvida para divulgação da **Maratona Feminina de Programação 2026 (MFP)**.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de substituir um panfleto tradicional por uma solução digital mais moderna, interativa e acessível.

A página reúne:
- Informações sobre o evento
- Datas importantes
- Regras da competição
- Categorias de participação
- Links de inscrição
- Contatos oficiais

---

## 🎯 Objetivo

Facilitar o acesso às informações da MFP 2026 e aumentar o alcance da divulgação por meio de um site responsivo integrado a QR Code.

---

## 👩‍🔬 Contexto

Projeto desenvolvido no âmbito do:

**MDC — Mulheres da Ciência**

Iniciativa voltada para a participação feminina na área de tecnologia e ciência.
